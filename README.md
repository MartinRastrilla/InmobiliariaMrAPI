# InmobiliariaMrAPI
API de Imboliaria para que Propietarios gestionen sus Inmuebles, Contratos y Pagos

## Base de Datos
- Base de datos en MySQL con XAMPP
- El archivo `.sql` de la base de datos está en la raíz del proyecto

## Configuración de IP
- Se estableció una IP determinada (que debe ser cambiada por la propia) para poder trabajar con la APP Mobile de Inmobiliaria
